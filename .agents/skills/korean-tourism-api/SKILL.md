# 한국관광공사 API

요구사항에 맞는 관광 정보·연관 관광지·무장애 여행·방문자 추이 API를 선택한다. 키는 환경 변수에서만 읽고 Git·Slack·Linear·PR에 기록하지 않는다. 외부 응답은 DTO로 격리하고, 실패·빈 결과를 서비스의 오류 계약에 맞게 처리한다.
